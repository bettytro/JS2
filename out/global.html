(constant) urlParams

this function checks if we have a user name in the query string (i.e we came from the feed or is on someone elses profile page) if we do, we fetch the user profile and posts for that user if not, we check if we have a user name in localStorage (i.e we came from the login page) if we do, we fetch the user profile and posts for that user if not, we log an error to the console this is because of reusability.
